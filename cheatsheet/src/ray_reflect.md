#### Ray Reflection
```
\       |       /
 \<--S--|---S->/
  Rin   |    Rout
   \    N    /
    \   |   /
     \ @|@ /
	  \ | /
	   \|/
------------------
```

Projection of Rin on N is Ncos(@), and all the rays are unit vectors
```
Rout = Ncos(@) + S
S = Ncos(@) - Rin
Rout = 2Ncos(@) - Rin = 2N(N . R) - Rin
```
